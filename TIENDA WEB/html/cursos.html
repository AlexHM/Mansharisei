Te ofrecemos:

Codeacademy

Curso de pago en el que puedes aprender de forma fácil y efectiva lenguajes como:
Html&CSS, Python, Javascript, Java, SQL, Ruby, C++, R, C#, PHP, entre otros.
El precio de la página web es de 13'99€ mensuales, pero existe la opción de comprar la versión para grupos, para la que el precio varía según la cantidad de alumnos.
Nuestra recomendación es probar la versión gratuita hasta donde la misma página permite y si te sientes plenamente cómodo/a aprendiendo a programar en ella, entonces una buena opción es la de pagar la suscripción mensual.

Enlace a Codeacademy

Programmr

Este es un sitio web gratuito donde puedes practicar cientos de actividades de java.
Se trata de completar las actividades sobre diferentes materias dentro del lenguaje Java. Cada vez que terminas una aparece un tick verde.
También puedes ver el proceso completado desde tu perfil.

Enlace a Programmr

Sololearn

Curso online que puedes llevar a cabo en casa con tu ordenador o en cualquier otro sitio con tu móvil.
Con este curso tienes la opción de aprender a programar en Python 3, C++, Java, JavaScript, C#, C, DS, HTML, PHP, CSS,... entre algunos otros.
En cuanto a su precio, es totalmente gratuito.
